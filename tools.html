Tool 02 // Chrome Extension

IntentLink v2

Google is a general-purpose engine. IntentLink makes it specific. It reads the intent behind your search — coding, medical, legal, finance — and routes you straight to the authoritative source instead of tutorial farms or ad-heavy results.

What's new in v2

  • 13 intent types (up from 9) — CODING, MEDICAL, LEGAL, FINANCE as first-class intents
  • Coding routes to MDN, Stack Overflow, GitHub, and official language docs
  • Medical routes to Mayo Clinic, NIH, and CDC
  • Legal routes to Cornell Law and Justia
  • Finance routes to Bogleheads and NerdWallet
  • News queries add a date filter + Reuters, AP, BBC
  • Shopping strips AliExpress / Temu and routes to Wirecutter & Reddit

Try these searches

  • react useeffect hook → MDN / Stack Overflow
  • lower back pain → Mayo Clinic / NIH
  • can my landlord enter without notice → Cornell Law
  • best budget keyboard → Wirecutter / Reddit
  • bitcoin news → Reuters / AP + date filter

Install

  1. Download the zip from GitHub and extract it
  2. Open chrome://extensions in Chrome
  3. Enable Developer mode (top-right toggle)
  4. Click Load unpacked and select the intentlink_v2 folder
